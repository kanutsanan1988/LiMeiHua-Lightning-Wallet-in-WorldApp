ความสำคัญของ Source Code ชุดนี้

LiMeiHua Source Code — พื้นฐานเทคโนโลยีสำหรับการพัฒนาโครงสร้างพื้นฐานของระบบการเงินในยุคสมัยใหม่

01
Bitcoin: บล็อกเชนที่แข็งแกร่งที่สุด
Bitcoin เป็น Blockchain ที่แข็งแกร่งที่สุดในโลก เราเลือกวาง transaction ทรัพย์สินของโครงการบนพื้นที่ที่ปลอดภัยที่สุดในโลก นั่นคือบล็อกเชนของ Bitcoin
02
Lightning Network สำหรับ UBI
เราเลือก Lightning Network เป็น Layer 2 ของ Bitcoin เพื่อแจกจ่าย Universal Basic Income (UBI) สำหรับมนุษย์เท่านั้น
03
WorldApp: ยืนยันความเป็นมนุษย์
เราใช้ WorldApp คัดกรองเฉพาะมนุษย์ (LiMeiHua Lightning Wallet in WorldApp, LiMeiHua Taproot Airdrop in WorldApp)
04
โอเพนซอร์สพื้นฐาน
source code เหล่านี้เป็น open source พื้นฐาน สำหรับ developer, บริษัท, ฟินเทค, Startup, องค์กรรัฐ นำไปพัฒนาต่อยอด

ชุด Source Code ทั้ง 9 ชุด

01
Create Taproot Assets
เว็บแอปพลิเคชันสำหรับสร้างและจัดการ Taproot Assets token บนบล็อกเชน Bitcoin ผ่าน Lightning Network
→ ดูบน GitHub → เอกสาร
02
Taproot Assets WordPress Plugin
ปลั๊กอิน WordPress สำหรับสร้างและจัดการ Taproot Assets token โดยตรงจากแดชบอร์ด WordPress
→ ดูบน GitHub → เอกสาร
03
Taproot All Bridge
สะพานข้ามเชนเชื่อมต่อ ERC-20 tokens (Ethereum) กับ Taproot Assets บน Bitcoin ผ่าน smart contracts
→ ดูบน GitHub → เอกสาร
04
Taproot Payment Gateway
ระบบชำระเงินสำหรับรับ Taproot Assets และ Lightning Network payments ในแอปพลิเคชันอีคอมเมิร์ซและธุรกิจ
→ ดูบน GitHub → เอกสาร
05
Taproot Airdrop Zone
แพลตฟอร์มสำหรับแจกจ่าย Taproot Assets tokens ผ่าน airdrops พร้อมการตรวจสอบสิทธิ์และจัดการการ claim
→ ดูบน GitHub → เอกสาร
06
Lightning Wallet
กระเป๋า Lightning Network หลายแพลตฟอร์ม (Web, Mobile, Desktop) สำหรับส่ง รับ Bitcoin และ Taproot Assets
→ ดูบน GitHub → เอกสาร
07
Lightning Wallet in WorldApp
กระเป๋า Lightning Network ที่เชื่อมต่อกับ WorldApp ใช้ World ID สำหรับยืนยันตัวตนมนุษย์และแจกจ่าย UBI
→ ดูบน GitHub → เอกสาร
08
Taproot Airdrop in WorldApp
ระบบ airdrop Taproot Assets ภายใน WorldApp แจกจ่าย tokens เฉพาะมนุษย์ที่ยืนยันผ่าน World ID
→ ดูบน GitHub → เอกสาร
09
Lightning Node
ระบบจัดการ Lightning Network node สำหรับ deploy ติดตาม และจัดการ LND/CLN nodes พร้อมรองรับ Taproot Assets
→ ดูบน GitHub → เอกสาร

Business Model แบบ Startup

โมเดลรายได้
  • ค่าบริการแจกจ่าย UBI
  • ตลาด mini-app WorldApp
  • ฟีเจอร์กระเป๋าพรีเมียม
  • เชื่อมต่อ World ID ระดับองค์กร
กลุ่มเป้าหมาย
  • ผู้ใช้ WorldApp (10M+)
  • ผู้ดูแลโปรแกรม UBI
  • NGOs และองค์กรช่วยเหลือ
  • โปรแกรมสวัสดิการรัฐบาล
คุณค่าที่นำเสนอ
  • กระเป๋าเดียวที่มี World ID + Lightning
  • แจกจ่าย UBI ป้องกัน Sybil
  • ไม่ต้องติดตั้งแอปแยก
  • เข้าถึงเฉพาะมนุษย์ที่ยืนยันแล้ว
ความได้เปรียบในการแข่งขัน
  • ระบบนิเวศ WorldApp (10M+ ผู้ใช้)
  • Biometric proof-of-personhood
  • ประสิทธิภาพ Lightning Network
  • กระเป๋า Lightning สำหรับ UBI แห่งแรก
